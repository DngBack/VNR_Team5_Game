# 🚀 BẮT ĐẦU TẠI ĐÂY

## Làm theo các bước sau để chạy Quiz Game:

### 1️⃣ Cài đặt
```bash
npm install
```

### 2️⃣ Chạy server
```bash
npm start
```

### 3️⃣ Mở browser
```
http://localhost:3000
```

### 4️⃣ Chọn vai trò
- **Admin**: Tạo phòng và quản lý game
- **Player**: Tham gia làm quiz
- **Single Player**: Chơi game gốc

## 📖 Hướng dẫn chi tiết
